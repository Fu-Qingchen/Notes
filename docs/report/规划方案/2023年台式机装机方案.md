# 2023年台式机选配方案

!!! note

    本次台式机的装机方案由研究生同学胡倚铭提供

## 需求

- 能够在未来 3 年在 2K 60Hz 下流畅运行原神应用
- 需要有较大的内存，能够安装多个 Docker 镜像
- 电脑需要体积小，便于安装
- 预期价位：￥5000 - 8000
